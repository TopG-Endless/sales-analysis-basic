# 📊 Sales Analysis Project
  
  📌 Objective

  Analyze product sales data to identify top-performing products and revenue distribution.
  
  🔧 Tools Used
  
  Google Sheets


  📊 Key Analysis

  Calculated total revenue per product
  Calculated percentage contribution
  Categorized products using ranking logic


  📈 Key Insights

  The Lapto generates ~56.5% of total revenue
  Sales are highly concentrated in one product
  Some products contribute very little
